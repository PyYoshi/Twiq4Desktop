Twiq for Desktop
================

Twiq is Twitter Client. Written in Python.

### Features

* Post: Post to Twitter
* Undo: Remove latest tweet
* Multiple Accounts

### Shortcut Keys

* Post: Ctrl+Enter(Return)
* Undo: Ctrl+U

### How to use?

Prepare:

```bash
$ mkvirtualenv Twiq
$ pip install -r requirements
$ ./build_resource.sh
```

Run:

```bash
$ ./twiq.sh
```

or

```bash
$ workon Twiq
$ python app.py
```
